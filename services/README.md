# VZW Services

## What This Directory Is

This directory contains the Verizon service set used by the deployment. These
services are Flask microservices generated by the internal catalog worker and
organized one service per directory.

## Who Should Read This

Read this if you need to inspect a specific service, compare service-level docs,
or understand how the Verizon deployment breaks its API surface into service
directories.
